Questions & Answers


Q: What CSV-encoding is used?
A: utf-8 & ascii are tested
Q: What is the layout of the CSV?
A: account number (IBAN-format) / transaction date (format 99-99-99) / valuta code - no validation / credit debet (C or D)/ amount / counter account / counter account holder / valuta date / payment method / description / payment type / authorization number / collector ID / address / reference / book date
Q: What is the Reference used for?
A: Reference is a unique identifier, accounts will be imported only, when a unique identifier has been provided
Q: Which banks are currently supported?
A: Currently the CSV-format used, is that of the Dutch bank KNAB.
